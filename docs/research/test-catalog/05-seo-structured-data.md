# SEO & Structured Data — Test Catalog

Provenance-first catalog for the LaunchAudit web-application launch-readiness auditor.
Every row traces to a real, named source surfaced by Perplexity (`sonar-pro`) and its
citations. Raw retrieval JSON is listed under **Raw evidence**. Standard ref = Lighthouse
SEO audit ID where one exists, otherwise the Google Search Central topic name (or the
canonical standard, e.g. Open Graph property, Core Web Vitals metric).
